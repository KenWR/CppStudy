# EX 01

## Description

iter 라는 3개의 파라미터들를 받고 아무것도 반환하지 않는 함수 템플릿을 구현하라.   

각 파라미터는 다음과 같은 의미를 가진다.      
1. 배열의 주소   
2. 배열의 길이    
3. 각 배열의 요소로 호출할 수 있는 함수   

충분한 테스트 실행을 해주는 main.cpp 파일이 주어진다.
iter 함수 템플릿은 모든 타입의 배열에서 동작하여야 하며 세 번째 파라미터는 함수 템플릿으로 인스턴스화 될 수 있어야 한다.
